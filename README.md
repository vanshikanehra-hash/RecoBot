# 🤖 RecoBot — AI Movie & Book Recommendation Bot

> Ask RecoBot anything about movies or books and get instant personalized recommendations powered by Groq LLM.

---

## What it does

RecoBot is a conversational AI that gives personalized movie and book recommendations based on your mood, preferences, or descriptions.

**Example queries:**
- "I want a mind-bending sci-fi movie"
- "Suggest a book about self improvement"
- "I love Christopher Nolan films"
- "Something emotional and deep"

---

## What I learned

- Loading and running a 125M parameter LLM on a GPU (NVIDIA T4)
- Fine-tuning a model on custom dataset using HuggingFace Transformers
- Why small models need large datasets (repetition problem)
- Prompt engineering with Groq API for production-quality results

---

## Tech Stack

- **Google Colab** — Free T4 GPU environment
- **HuggingFace Transformers** — Model loading and fine-tuning
- **Groq API** (llama-3.1-8b-instant) — LLM inference
- **PyTorch** — Deep learning framework
